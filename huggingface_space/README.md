# GazeAlign

Gaze-supervised medical image classification. Upload an image, click to
lay down fixation points (simulating where a radiologist looked), and see
the model's predicted class alongside its learned gaze-conditioned
attention mask.

See the [GitHub repository](https://github.com/MohammedOussamaBEN/GazeAlign)
for training code, evaluation scripts, and the paper.
